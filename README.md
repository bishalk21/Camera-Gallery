# Camera + Gallery Application (Loom)

- Heavy Browser APIs Usage
  - Media Stream API
  - Media Recorder API
  - Media Devices API
  - navigator API
- Uses IndexedDB for storing recorded videos
- Canvas for video preview

# Features

- Image capture
- Video recording
- Gallery to view recorded videos (in browser)
